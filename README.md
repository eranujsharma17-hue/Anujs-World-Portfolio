# Anuj's World - AI Capabilities Portfolio

A premium, high-performance portfolio website showcasing AI-related capabilities and projects.

## Tech Stack
- HTML5 / CSS3 (Vanilla)
- JavaScript (ES6+)
- GitHub Pages for Deployment

## Features
- **Modern AI Aesthetic**: Dark mode with neon glow and glassmorphism.
- **Dynamic Projects**: Interactive project cards with hover effects.
- **Responsive Design**: Optimized for all devices.
- **Smooth Animations**: Intersection observer based fade-ins.
